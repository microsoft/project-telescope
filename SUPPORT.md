# Support

## How to file issues and get help

This project uses [GitHub Issues](https://github.com/microsoft/project-telescope/issues) to track
bugs and feature requests. Please search the existing issues before filing new issues to avoid
duplicates.

For help and questions about using this project, please open a
[GitHub Discussion](https://github.com/microsoft/project-telescope/discussions).

## Microsoft Support Policy

Support for this project is limited to the resources listed above.
